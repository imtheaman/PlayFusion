## Description

Enables quick web searches directly from command line using various search engines without switching to browser.  
[GitHub](https://github.com/ohmyzsh/ohmyzsh/blob/master/plugins/web-search/web-search.plugin.zsh)

## Essential Commands

- `google [query]` - Search Google 
- `bing [query]` - Search Bing
- `yahoo [query]` - Search Yahoo
- `ddg [query]` - Search DuckDuckGo
- `github [query]` - Search GitHub repositories

## Advanced Commands

- `stackoverflow [query]` - Search Stack Overflow
- `scholar [query]` - Search Google Scholar
- `archive [url]` - Search Wayback Machine
- `baidu [query]` - Search Baidu

## Usage Examples

```bash
# Programming searches
google "javascript promises tutorial"
github "react component library"
stackoverflow "zsh command not found"
scholar "machine learning algorithms"

# Open search homepage (no arguments)
google              # Opens google.com
github              # Opens github.com

# Custom search engines (in .zshrc)
ZSH_WEB_SEARCH_ENGINES=(
  reddit "https://www.reddit.com/search/?q="
  npm "https://www.npmjs.com/search?q="
)
```
