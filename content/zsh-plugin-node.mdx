## Description

Provides utilities for Node.js development, primarily offering quick access to Node.js documentation.  
[GitHub](https://github.com/ohmyzsh/ohmyzsh/blob/master/plugins/node/node.plugin.zsh)

## Essential Commands

- `node-docs` - Opens full Node.js API documentation
- `node-docs <section>` - Opens specific section documentation

## Advanced Usage

- Version-specific documentation detection
- Cross-platform browser compatibility
- Module-specific documentation access

## Usage Examples

```bash
# Open full Node.js documentation
node-docs

# Open specific module documentation
node-docs http
node-docs fs
node-docs crypto
node-docs stream
```
