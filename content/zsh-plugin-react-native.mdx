## Description

Provides aliases and autocompletion for React Native CLI commands, supporting both iOS and Android development workflows.  
[GitHub](https://github.com/ohmyzsh/ohmyzsh/blob/master/plugins/react-native/README.md)

## Essential Commands

- `rn` → `react-native`
- `rns` → `react-native start`
- `rnios` → `react-native run-ios`
- `rnand` → `react-native run-android`
- `rnlink` → `react-native link`

## iOS Simulator Shortcuts

- `rnios6` → `react-native run-ios --simulator "iPhone 6"`
- `rnios8` → `react-native run-ios --simulator "iPhone 8"`
- `rnios8p` → `react-native run-ios --simulator "iPhone 8 Plus"`
- `rniosse` → `react-native run-ios --simulator "iPhone SE"`

## Advanced Commands

- `rnland` → `react-native log-android`
- `rnlios` → `react-native log-ios`

## Usage Examples

```bash
# Development workflow
rns                   # Start Metro bundler
rnios                # Run on default iOS simulator
rnand                # Run on Android emulator

# Device-specific testing
rnios6               # Test on iPhone 6 simulator
rnios8p              # Test on iPhone 8 Plus

# Debugging
rnland               # Monitor Android logs
rnlios               # Monitor iOS logs
```
